# 2D Python Side-Scroller Game

## Overview
A fully playable 2D side-scrolling game built using the Python Arcade library. Includes movement, collision detection, scoring, and level progression.

## Features
- Player movement and physics
- Enemy behavior and collisions
- Score tracking
- Multi-level progression

## Technologies Used
- Python
- Arcade Library
- Object-Oriented Programming (OOP)
